# 3월 탭 설계 (PLAN_MARCH_TAB)

## 1. 개요

- **목표**: 1·2월에 이어 **3월 탭**을 추가하고, 3월 전용 시상 구조를 반영한다.
- **데이터**: `performance["2026-03"]`, 3월 1주차 실적(`weekly.week1` 등) — 기존 MC_LIST/업로드 구조에서 202603 회차 반영 시 동일하게 확장 가능.
- **UI**: 월 선택 드롭다운에 **3월** 추가 (`selectedViewMonth`: `1 | 2 | 3`).

---

## 2. 구간 박스 뱃지 (공통 UI) — 모든 구간 시상 적용

**모든 구간 시상**에서 달성 진행을 한눈에 보이도록 **중간에 구간 박스 뱃지**를 넣는다.

- **표시 형식**: 구간을 만원 단위로 나열. 예: `10 / 20 / 30 / 50 / 100`
  - 시상별 실제 구간에 맞게 숫자만 변경 (예: 1주차 특별은 20/30/50/80/100/120, 파타야는 40/60/80/100 등).
- **달성한 구간**: **금색(gold)** 으로 표시 — 배경 또는 테두리/텍스트를 금색 계열로 강조.
- **다음 도전 구간**: **레드(red)** 로 표시 — “이 구간 달성 시 다음 시상”임을 강조해 달성 독려.
- **미달성 구간**: 회색 또는 기본색으로 표시 (선택).
- **적용 대상**: 1주차 특별 현금시상, 1주차 PATAYA 특별, 파타야 여행시상, 2배 메리츠클럽, 메리츠클럽+ 등 **구간이 있는 모든 시상 카드**.

---

## 3. 3월 시상 목록 및 규칙

### 3.1 파타야 여행시상 (3월 전용 — 4·5월은 추후)

- **우선 3월만 반영**: 당분간 4·5월 데이터/합산은 구현하지 않고, **3월 실적만**으로 달성 구간을 보여 주며 달성 독려.
- **구간 표시**: **40 / 60 / 80 / 100** (만원) — 카드 중간에 구간 박스 뱃지로 배치.
  - LCC 40만 → 1박추가 60만 → 1박+국적기+5성 80만 → 1박+1인실+국적기 100만.
- **구간 박스 뱃지 적용**
  - 달성한 구간: **금색** 표시.
  - 다음 구간: **레드** 표시 (다음 목표 달성 독려).
- **데이터**: `performance["2026-03"]` (3월 실적)만 사용. 4·5월 연속/합산·CHAMP는 추후 확장 시 추가.

**내부 상수 (3월만)**  
- 파타야 3월 구간(원): 400_000, 600_000, 800_000, 1_000_000.

---

### 3.2 1주차 특별 현금시상 (3월 1주차)

| 실적 (만원) | 시상금 (만원) |
|-------------|----------------|
| 20 | 20 |
| 30 | 30 |
| 50 | 100 |
| 80 | 240 |
| 100 | 400 |
| 120 | 600 |

- **적용**: 3월 1주차 실적만 사용.  
- **티어 배열 (원 단위)**:  
  `MAR_W1_SPECIAL: [[1200000, 6000000], [1000000, 4000000], [800000, 2400000], [500000, 1000000], [300000, 300000], [200000, 200000]]`
- **구간 박스 뱃지**: 카드 중간에 **20 / 30 / 50 / 80 / 100 / 120** (만원) 표시. 달성 구간 **금색**, 다음 구간 **레드**.

---

### 3.3 1주차 PATAYA 특별 현금시상 (3월 1주차)

| 실적 (만원) | 시상금 (만원) |
|-------------|----------------|
| 20 | 20 |
| 30 | 30 |
| 50 | 100 |
| 70 | 210 |
| 100 | 500 |

- **적용**: 동일하게 3월 1주차 실적.  
- **티어 배열 (원 단위)**:  
  `MAR_W1_PATAYA: [[1000000, 5000000], [700000, 2100000], [500000, 1000000], [300000, 300000], [200000, 200000]]`
- **구간 박스 뱃지**: **20 / 30 / 50 / 70 / 100** (만원) 표시. 달성 **금색**, 다음 **레드**.
- **UI**: 1주차 카드는 2개로 둘 수 있음.  
  - "1주차 특별 현금시상" (3.2)  
  - "1주차 PATAYA 특별 현금시상" (3.3)  
  또는 한 카드 안에 두 표를 구분해 배치.

---

### 3.4 2배 메리츠클럽

- **규칙**: 기존과 동일 — 전월(2월)·당월(3월) 각 20만원 이상 시.  
- **데이터**: `performance["2026-02"]`, `performance["2026-03"]`.  
- **구간 박스 뱃지**: 20만 이상 달성 여부를 구간 뱃지로 표시할 경우, 달성 **금색**, 미달성/다음 **레드** 느낌으로 적용 가능.

---

### 3.5 메리츠클럽+

- **규칙**: 기존과 동일 — min(전월, 당월) 구간 달성 후 당월 실적로 다음 구간 진행.  
- 3월 탭: 전월=2월, 당월=3월.  
- **구간 박스 뱃지**: 20/40/60/80/100만 등 구간이 있으면 동일하게 **금색(달성) / 레드(다음 구간)** 적용.

---

### 3.6 2월 정규 시상

- **의미**: 3월 탭에서 "지난달(2월)에 이미 받은/받을 정규 시상" 요약.
- **표시**: 카드 1장 — "2월 정규 시상" 제목, 2월 실적·2월 정규 시상금(기존 2월 로직으로 계산한 값) 요약.
- **역할**: 3월 시상과 구분해, 2월 달성분을 한곳에 모아 보여주기.
- (구간이 있으면 동일하게 구간 박스 뱃지·금색/레드 적용 가능.)

## 4. UI/화면 구성 제안

### 4.1 월 선택

- 드롭다운: **1월 | 2월 | 3월** (`selectedViewMonth`: `1 | 2 | 3`).
- 3월 선택 시:
  - 랭킹/정렬 키: `rankKeyMonth = "2026-03"`.
  - 당월 실적: `performance["2026-03"]`, 전월: `performance["2026-02"]`.
  - 1주차: 3월 1주차 실적 사용 (특별 + PATAYA 둘 다).

### 4.2 비파트너 3월 시상 카드 배치 (제안 순서)

1. **파타야 여행시상** (3월 전용) — 구간 **40 / 60 / 80 / 100** (만원) 박스 뱃지, 달성 금색·다음 레드.
2. **1주차 특별 현금시상** — 구간 20/30/50/80/100/120 뱃지.
3. **1주차 PATAYA 특별 현금시상** — 구간 20/30/50/70/100 뱃지.
4. **2배 메리츠클럽** — 구간 뱃지 적용 시 금색/레드.
5. **메리츠클럽+** — 구간 뱃지 적용 시 금색/레드.
6. **2월 정규 시상** — 2월 달성분 요약.

### 4.3 파타야 카드 상세 디자인 (3월 전용)

- **상단**: "파타야 여행시상 (3월)".
- **중간 구간 박스 뱃지**: **40 / 60 / 80 / 100** (만원) — 3월 실적 기준.
  - **달성한 구간**: 금색(gold) 배경 또는 테두리/텍스트.
  - **다음 구간**: 레드(red)로 표시해 "이 구간 달성하면 다음 등급" 달성 독려.
- **현재 3월 실적** 표시 + 다음 목표(40→60→80→100만) 문구로 달성 독려.
- **데이터**: `performance["2026-03"]` 만 사용. (4·5월·합산·CHAMP는 추후 확장.)

## 5. 데이터·API 확장

| 항목 | 내용 |
|------|------|
| **performance** | `2026-03` 키 추가. (이미 RANK_MONTHS 등에 03 있으면 유지, 없으면 추가.) |
| **weekly** | 3월 선택 시 1주차 = 3월 1주차 실적. (MC_LIST에 202603 회차가 오면 동일 인덱스로 주차 실적 확장.) |
| **랭킹** | 3월 선택 시 `rankKeyMonth = "2026-03"`, 정렬/ranks에 2026-03 반영. |
| **실적 추이 차트** | 3월 막대 추가 시 "3월" 클릭 시 3월 탭으로 전환 가능하도록 확장. |
